Python 3D raycaster (A Level project):

A 3D renderer that uses raycasting to create a 3D scene using a 2D map. It was developed using Python and Pygame, and I've compiled it to WebAssembly so it can also be played in the browser (note: browser version will have slower speeds than original Python).




Gravity/sand simulator:

A cellular automaton Python project that lets you simulate sand in gravity. Actually quite relaxing.








Game for the original GameBoy console:

I wrote a simple game in C that compiled into a ROM for the original GameBoy. It can be played online in a broswer emulator, or be written to a flashcart and be played on an actual GameBoy. A fun embedded project!
